1. Google Fonts Link

Place in <head> of EVERY page. This is the ONLY approved font loading method.

<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Open+Sans:ital,wght@0,300;0,400;0,600;0,700;0,800;1,400&family=Zilla+Slab:ital,wght@0,300;0,400;0,500;0,600;0,700;1,400&display=swap" rel="stylesheet">
BANNED fonts: Barlow, Manrope, JetBrains Mono, Inter. If you see these in a Google Fonts link, replace with the above.
